# python-not-midterm
### Get Started
```
django-admin startproject midterm_project
cd midterm_project
python3 manage.py migrate
python3 manage.py runserver
```
see `localhost:8000`

### Create Admin User
```
python3 manage.py createsuperuser --username=your_username --email=your_email@gmail
```
see `localhost:8000/admin`
